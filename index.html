Maps to Mathematics for Machine Learning

This track is the interactive, intuition-first front end to the standard graduate text Mathematics for Machine Learning by Deisenroth, Faisal & Ong (Cambridge University Press, 2020). Each session links to the exact sections there for the formal treatment; every lesson carries a 📖 reference and optional "Going deeper" proofs. The book's PDF is free for personal use.
