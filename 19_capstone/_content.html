SynthesisOne example that uses (almost) everything

Training a spam classifier — mathematically. Follow one email through the whole pipeline and watch the sessions light up:

S1, S13
The email becomes a feature vector $\mathbf{x}\in\mathbb{R}^n$ (word counts / an embedding) — a point in space. Its label is a Bernoulli outcome (spam = 1). ↳ S1 ↳ S13
S2, S4
The model computes a score $\mathbf{w}\cdot\mathbf{x}+b$ — a dot product — and for a batch of emails that's one matrix multiplication $XW^\top$. A deep model stacks several (composition). ↳ S2 ↳ S4
S13, S17
A sigmoid squashes the score to $q=P(\text{spam})$ (a Bernoulli parameter). The loss is cross-entropy = negative log-likelihood, derived by MLE from the Bernoulli assumption. ↳ S17
S18
That cross-entropy is a KL divergence from the model's prediction to the one-hot label — how many extra bits the model's beliefs cost. ↳ S18
S8–S11
Training minimizes that loss: backprop (the chain rule run backward) computes the gradient w.r.t. every weight, and gradient descent steps opposite it, one mini-batch at a time. ↳ S10 ↳ S11
S1, S17
To fight overfitting, add L2 regularization — which is a Gaussian prior on the weights (MAP), and geometrically an L2 norm penalty. ↳ S17
S5, S7, S14
Before training you might standardize features (mean/variance, S14), drop collinear ones (rank, S5), or PCA-compress them (SVD, S7). ↳ S5 ↳ S7
S12, S15, S16
Evaluating: precision on the rare spam class is base-rate-sensitive (Bayes, S12); the test-set metric is a sample mean with a standard error (CLT, S15); and "is model B better than A?" is a hypothesis test / A-B (S16). ↳ S16

Twelve of the eighteen sessions, in one classifier. That's the point: these aren't separate subjects — they're the stages of a single machine that turns data into decisions.

RevisionInterview cheat-sheet

The one-liners worth having on instant recall, by pillar.

IdeaThe one-liner
Vector / normAn ordered list = a point in ℝⁿ; ‖v‖ is its length; distance = ‖a−b‖.
Dot producta·b = ‖a‖‖b‖cos θ — agreement; cosine similarity normalizes out length.
MatrixA linear transformation; its columns are where the basis vectors land.
MatmulComposition ("B then A"); not commutative, is associative; entry = row·column.
Rank / detIndependent columns; full rank ⟺ det≠0 ⟺ invertible ⟺ spans ℝⁿ.
EigenAv=λv: directions a matrix only scales; symmetric ⇒ orthogonal eigenvectors.
SVD / PCAAny matrix = rotate·stretch·rotate; PCA = eigenvectors of the covariance.
Derivative / gradientSlope; ∇f points steepest uphill, so −∇f is downhill.
Chain rule / backpropDerivatives multiply along a chain; backprop = it run backward (upstream × local).
Gradient descentw ← w − η∇L; η too big diverges, too small crawls; mini-batch is the default.
BayesPosterior ∝ likelihood × prior; always respect the base rate.
Gaussianμ (center), σ (spread); 68–95–99.7; covariance ellipse = PCA axes.
Variance / covarianceVar=E[X²]−E[X]²; correlation∈[−1,1]; uncorrelated ≠ independent.
CLTSample means → Gaussian, SE = σ/√n; halve error ⇒ 4× data.
Inferencep = P(data | H₀), not P(H₀ | data); report effect size + CI.
MLEMaximize likelihood = minimize NLL; Gaussian→MSE, Bernoulli→cross-entropy.
Info theoryCross-entropy = entropy + KL; minimizing CE = fitting q to p; KL is asymmetric.
